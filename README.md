# tourism-management-system-2
此旅游管理系统基于springboot+vue开发。后端springboot负责旅游资源管理、订单处理、用户信息存储等业务逻辑，保障系统稳定运行。前端vue则提供了直观的用户界面，具有良好的响应性。用户可以在该系统中浏览旅游景点、预订行程、管理订单，也可查看旅游攻略，为游客和旅游从业者提供便捷的旅游服务体验，促进旅游业的发展。
